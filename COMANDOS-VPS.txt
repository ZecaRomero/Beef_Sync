=== COMANDOS PARA EXECUTAR NO VPS (SSH) ===

--- 0. PREPARAÇÃO DO AMBIENTE (Se for a primeira vez) ---
# 1. Instalar Docker (se não tiver)
curl -fsSL https://get.docker.com -o get-docker.sh
sh get-docker.sh

# 2. Criar diretório
mkdir -p /opt/beefsync
cd /opt/beefsync

# 3. Subir Banco de Dados (Postgres)
# Certifique-se de ter o arquivo docker-compose.yml (o principal, com o postgres)
docker compose up -d postgres

# 4. Criar rede se necessário (normalmente o docker compose cria)
docker network create beef_sync_default 2>/dev/null || true

--- 1. DEPLOY DO APP ---
# 1. Ir para o diretório
cd /opt/beefsync

# 2. Parar e remover container antigo
docker stop beefsync-app 2>/dev/null; docker rm beefsync-app 2>/dev/null

# 3. Build e subir
docker compose -f docker-compose.beefsync-build.yml build --no-cache
docker compose -f docker-compose.beefsync-build.yml up -d

--- 2. MIGRAR DADOS (Opccional) ---
# Rode no SEU COMPUTADOR (PowerShell):
# npm run migrar:vps

--- 3. Verificar logs ---
docker logs beefsync-app --tail 50
